<h2>Mod Description</h2>
<p style="text-align:center"><a href="https://serilum.com/" rel="nofollow"><img src="https://cdn.serilum.com/description/header/header.png" alt="" width="838" height="400"></a></p>
<p style="text-align:center"><a href="https://curseforge.com/members/serilum/projects" target="_blank" rel="nofollow"><img src="https://workflow.serilum.com/badges/svg/curseforge.svg" width="200"></a> <a href="https://modrinth.com/user/Serilum" target="_blank" rel="nofollow"><img src="https://workflow.serilum.com/badges/svg/modrinth.svg" width="200"></a> <a href="https://patreon.com/serilum" target="_blank" rel="nofollow"><img src="https://workflow.serilum.com/badges/svg/patreon.svg" width="200"></a> <a href="https://youtube.com/@serilum" target="_blank" rel="nofollow"><img src="https://workflow.serilum.com/badges/svg/youtube.svg" width="200"></a></p>
<p><strong><span style="font-size:24px">Requires the library mod&nbsp;<a style="font-size:24px" href="https://curseforge.com/minecraft/mc-mods/collective" target="_blank" rel="nofollow">Collective</a>.<br></span></strong></p>
<p><span style="font-size:18px">Adventure Mode Tweaks makes various changes to, you guessed it, adventure mode game mechanics. The idea came from realizing that players are able to break paintings, which obviously shouldn't be the case. The functionality was extended from there.<br><br>For example, without the mod, you are able to break minecarts but not place them. And both break and place boats. Breaking leads can be unintended by RPG map makers as well. Everything is configurable so you can choose exactly what to enable and disable.<br><br>You can also use the mod to have all new players spawn in Adventure Mode. Useful if you have a mod pack that uses a specific scenario. See the features and images below for more information!<br><br><strong><span style="font-size:24px">Features:</span> (while in Adventure Mode)</strong><br>&nbsp;&bull;&nbsp; Everything is entirely configurable.<br>&nbsp;&bull;&nbsp; Force Adventure Mode in new worlds, includes UI updates.<br>&nbsp;&bull;&nbsp; Prevent passive and/or hostile mob attacks.<br>&nbsp;&bull;&nbsp; Prevent users from breaking paintings.<br>&nbsp;&bull;&nbsp; Prevent item frames from being broken, but allow taking items.<br>&nbsp;&bull;&nbsp; Both place and break boats and minecarts (or disable it instead).<br>&nbsp;&bull;&nbsp; Prevent entity interaction entirely.<br>&nbsp;&bull;&nbsp; Prevent breaking and using leads.<br>&nbsp;&bull;&nbsp; Prevent the use of ender pearls.<br>&nbsp;&bull;&nbsp; Disable being able to sleep in beds.<br>&nbsp;&bull;&nbsp; Disable fishing and/or hooking entities.<br>&nbsp;&bull;&nbsp; Prevent shearing sheep and/or milking cows.<br></span><span style="font-size:18px">&nbsp;&bull;&nbsp; Prevent placed TNT from being lit.</span><br><br><br><strong><span style="font-size:24px">Configurable:</span> <span style="color:#008000;font-size:14px"><a style="color:#008000" href="https://github.com/Serilum/.information/wiki/how-to-configure-mods" target="_blank" rel="nofollow">(&nbsp;how do I configure?&nbsp;)</a></span><br></strong></p>
<div class="spoiler">
<p><span style="font-size:12px"><strong>forceSurvivalToAdventureMode</strong>&nbsp;(default = false): Globally changes Survival Mode to Adventure Mode. This is done when a player joins a world. The Create World Screen UI is updated accordingly as well.</span><br><br><span style="font-size:12px"><strong>preventAttackingCreatureMobs</strong>&nbsp;(default = false): Stops players in Adventure Mode from harming passive mobs.</span><br><span style="font-size:12px"><strong>preventAttackingMonsterMobs</strong>&nbsp;(default = false): Prevents Adventure Mode players from attacking hostile mobs.</span><br><span style="font-size:12px"><strong>preventEntityInteraction</strong>&nbsp;(default = false): Disables right-click interactions with entities while in Adventure Mode.</span><br><br><span style="font-size:12px"><strong>preventDestroyingPaintings</strong>&nbsp;(default = true): Adventure Mode players can't break paintings. In vanilla Minecraft this is allowed.</span><br><span style="font-size:12px"><strong>preventDestroyingItemFrames</strong>&nbsp;(default = true): Prevents breaking item frames when in Adventure Mode. In vanilla Minecraft this is allowed.</span><br><span style="font-size:12px"><strong>preventLootingItemFrames</strong>&nbsp;(default = false): Stops Adventure Mode players from taking items from item frames.</span><br><br><span style="font-size:12px"><strong>preventBreakingMinecarts</strong>&nbsp;(default = false): Prevents players in Adventure Mode from destroying minecarts.</span><br><span style="font-size:12px"><strong>allowPlacingMinecarts</strong>&nbsp;(default = true): Allows minecarts to be placed even in Adventure Mode. Default Minecraft behaviour is no placement.</span><br><span style="font-size:12px"><strong>preventBreakingBoats</strong>&nbsp;(default = false): Prevents players in Adventure Mode from destroying boats.</span><br><span style="font-size:12px"><strong>allowPlacingBoats</strong>&nbsp;(default = true): Allows boat placement while in Adventure Mode. Default Minecraft behaviour is allowed placement.</span><br><br><span style="font-size:12px"><strong>preventCreeperBlockDamage</strong>&nbsp;(default = true): Stops creepers from destroying blocks when exploded near an Adventure Mode player.</span><br><span style="font-size:12px"><strong>preventUseOfEnderPearls</strong>&nbsp;(default = false): Blocks ender pearl from being used by Adventure Mode players.</span><br><span style="font-size:12px"><strong>preventBedSleeping</strong>&nbsp;(default = false): Prevents sleeping in beds while in Adventure Mode.</span><br><span style="font-size:12px"><strong>preventLightningTNT</strong>&nbsp;(default = false): Prevents Adventure Mode players from lighting TNT with a flint &amp; steel.</span><br><br><span style="font-size:12px"><strong>preventFishing</strong>&nbsp;(default = false): Adventure Mode players can't use fishing rods.</span><br><span style="font-size:12px"><strong>preventFishingRodHookedEntities</strong>&nbsp;(default = true): Stops players with fishing rods from hooking entities in Adventure Mode.</span><br><br><span style="font-size:12px"><strong>preventUseOfLeads</strong>&nbsp;(default = false): Prevent using and breaking leads in Adventure Mode.</span><br><span style="font-size:12px"><strong>preventCowMilking</strong>&nbsp;(default = false): Prevents cows from being milked while in Adventure Mode. Redundant if 'preventEntityInteraction' is enabled.</span><br><span style="font-size:12px"><strong>preventSheepShearing</strong>&nbsp;(default = false): Stops sheep from being sheared by Adventure Mode players. Redundant if 'preventEntityInteraction' is enabled.</span></p>
</div>
